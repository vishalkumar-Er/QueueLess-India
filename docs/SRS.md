# QueueLess India

# Software Requirement Specification (SRS)

---

## 1. Introduction

QueueLess India is a smart digital queue management application.

This application helps users avoid long waiting lines by allowing them to join virtual queues from their mobile phones.

---

## 2. Problem Statement

People waste a lot of time standing in queues at hospitals, banks, salons, temples, government offices and other public places.

There is no single platform where users can check live queue status before visiting.

---

## 3. Objective

To build a smart queue management platform that allows users to book virtual tokens, track live queue status and receive notifications before their turn.
